# EZ-AISurf 课程评分仪表盘

这是一个基于 Streamlit 开发的课程评分分析仪表盘，用于展示和分析课程评分数据。

## 功能特点

- 课程评分趋势分析
- 教师评分分布
- 学生评分分布
- 课程评分分布
- 评分明细数据展示
- 多维度数据筛选

## 数据文件

- `course_ratings_compact.json`: 课程评分数据
- `teachers.json`: 教师信息
- `students.json`: 学生信息

## 运行方式

```bash
streamlit run rating_dashboard.py
```
